# Picture-in-Picture (PiP) Ad Guidelines

This document contains reference guidelines and mandatory test case checks for PiP Ads.

## 1. General Rules
- Ad must collapse into a floating window over the app UI when the trigger condition is met.
- The PiP window must not block critical user flows.

## 2. Configuration Keys
- `positionShowPiP`: Position should match the configured value (e.g., top-left, bottom-right).
- Users should be able to drag or close the PiP window (if configured to allow).
